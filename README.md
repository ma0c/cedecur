## Local Usage
Be sure you have python3.7+ installed

execute

```bash
python3 -m venv cedecur_env
source ./cedecur_env/bin/activate

# or in windows
# cedecur_env\Scripts\activate.bat

# Then install the requirements
pip install -r requirements.txt

```
